# HA Energy Dashboard

An advanced Home Assistant dashboard focused on photovoltaic systems.

This project provides a complete set of dashboards, templates and sensors to monitor not only energy production and consumption, but also the real economic impact of a solar installation.

## ✨ Features

- 📊 Real-time photovoltaic monitoring
- 🔋 Battery status and statistics
- ⚡ Grid import/export
- 💰 Estimated electricity bill
- 📈 Monthly and yearly savings
- ☀️ Self-sufficiency calculation
- 📅 Annual energy summary
- 🗄️ SQL historical data reconstruction
- 🎨 Modern and responsive dashboard
